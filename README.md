## 🎯 Sobre o Projeto

TimeQuest é um sistema interativo de gestão de tempo que transforma sua rotina em uma jornada gamificada. Cada tarefa completada rende experiência (XP) e ajuda você a subir de nível, tornando a produtividade mais divertida e engajadora.

### ✨ Funcionalidades

- 📅 **Gerenciamento de Horários**: Adicione, edite e remova compromissos
- 🎮 **Sistema Gamificado**: Ganhe XP ao concluir tarefas
- 📈 **Progressão**: Suba de nível e acompanhe seu streak
- 🏆 **Dashboard**: Visualize estatísticas em tempo real
- 💾 **Persistência**: Dados salvos localmente ou no Neo4j

### 🛠️ Tecnologias

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Banco de Dados**: Neo4j (Graph Database)
- **Hospedagem**: Vercel
- **Estilização**: CSS com design system próprio

## 🚀 Como Executar

### Pré-requisitos

- Node.js (v14 ou superior)
- NPM ou Yarn
- Neo4j (local ou cloud)

### Instalação

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/timequest.git
cd timequest

# Instale as dependências
npm install

# Execute o servidor de desenvolvimento
npm start
